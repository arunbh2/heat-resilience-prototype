# Heat Resilience Decision Engine Prototype

This is a static browser-based prototype that can be hosted on GitHub Pages.

It uses demonstration ward-level data to rank heat risk, suggest intervention packages, estimate rough cost bands, and generate a first-draft municipal memo.

## How to use

Open `index.html` in any browser.

No Python, server, or installation is needed for the web version.

## How to host

Upload `index.html` and `README.md` to a public GitHub repository.

Then go to Settings → Pages → Deploy from a branch → main → /(root) → Save.

Your site will usually appear after a few minutes at:

`https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Important

This uses demonstration data only. Replace the sample data with real city/ward data before using it for decision-making.
